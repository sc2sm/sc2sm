Twitter/X Integration

Choose how you want to share your posts on Twitter/X.

Option A: URL Generation (Recommended)

Generate shareable Twitter URLs that open the compose window with your content pre-filled.

Creates a URL like: https://twitter.com/intent/tweet?text=...

Option B: Direct API Posting

Automatically post directly to Twitter using API credentials.

Requires Twitter Developer account and API credentials.

API Setup Instructions:
  1. Create a Twitter Developer account
  2. Create a new app and generate API keys
  3. Add the credentials to your .env file
  4. Restart the application

When enabled, approved posts will be automatically published to Twitter.
